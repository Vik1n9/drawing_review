# 各類場所消防安全設備設置標準

> 修正日期：民國 113 年 04 月 24 日

---

## 第三編 消防安全設計

### 第三章 避難逃生設備

#### 第一節 標示設備

##### 第 146 條

下列處所得免設出口標示燈、避難方向指示燈及避難指標：
- 一、主要為通往室外之走廊或出入口，且有自然採光，其照度達五十勒克司（Lux）以上者。
- 二、通往直通樓梯間之走廊或通道，未設有開口部，且無造成步行障礙之物品者。
- 三、洗手間、浴室、機械室、鍋爐房及其他類似處所。
- 四、自居室任一點易於識別該居室出入口者。
- 五、室內停車場，自任一點易於識別該層出入口者。
- 六、避難層對外出入口，設有直通之斜坡道者。
- 七、主要出入口至避難層對外出入口之步行距離在五十公尺以下之地下通道。
- 八、居室樓地板面積在二百平方公尺以下，且自居室任一點至出入口之步行距離在三十公尺以下者。
- 九、其他經中央主管機關公告之場所得免設。

##### 第 146-1 條

1. 出口標示燈及避難方向指示燈，依其設置場所之空間高度（縱向尺度）區分等級，其光度應符合下表規定：

| 等級 | 縱向尺度（空間高度） | 出口標示燈光度 | 避難方向指示燈光度 |
|------|---------------------|--------------|------------------|
| A 級 | 八公尺以上         | 五十燭光以上  | 六十燭光以上      |
| B 級 | 四公尺以上未滿八公尺 | 三十燭光以上  | 四十燭光以上      |
| C 級 | 未滿四公尺          | 十燭光以上    | 十五燭光以上      |

2. 居室面向避難方向指示燈設置方向之步行距離在避難方向指示燈有效範圍內者，得免設出口標示燈。
3. 前項有效範圍，指依空間高度（縱向尺度）區分等級之避難方向指示燈，其燈光能直接到達之步行距離範圍。

##### 第 146-2 條

出口標示燈及避難方向指示燈之有效範圍（步行距離），應依下表規定：

| 等級 | 出口標示燈有效範圍 | 避難方向指示燈有效範圍 |
|------|------------------|---------------------|
| A 級 | 四十公尺以下      | 二十公尺以下         |
| B 級 | 三十公尺以下      | 十五公尺以下         |
| C 級 | 二十公尺以下      | 十公尺以下           |

##### 第 146-3 條

出口標示燈及避難方向指示燈之設置位置，應符合下列規定：
- 一、出口標示燈：
    - （一）應設於避難層之各直通樓梯、通往直通樓梯之出入口、通往地面層之斜坡道及室內走廊之直接對外出入口。
    - （二）裝設位置應設於設置場所之出入口上方，無開口部之天花板時得設於牆面。
    - （三）避難層之各出入口以同一廣告招貼標示，且不妨礙辨認者，得免設。
- 二、避難方向指示燈：
    - （一）應設於通往直通樓梯或避難層之走廊、通道、樓梯及斜坡道之牆面或天花板。
    - （二）自居室任一點至避難方向指示燈之步行距離，不得超過其有效範圍。
    - （三）於樓梯間或斜坡道內，應設置於各層樓梯平台或斜坡道之轉折點。

##### 第 146-4 條

出口標示燈及避難方向指示燈之裝設，應符合下列規定：
- 一、設於天花板或天花板附近時，應不突出於天花板下方且不與其他設備互相妨礙。
- 二、出口標示燈設於出入口上方時，其下端距樓地板面之高度，在二點五公尺以下。
- 三、離天花板高度在二點五公尺以上之出口標示燈，其下方設有移動式隔間牆等障礙物時，應於其下方增設之。
- 四、除使用發光二極體（LED）為光源者外，燈具周圍溫度不得超過攝氏四十度。
- 五、避難方向指示燈設於樓梯間時，應設於各層樓梯平台或其上方牆面。
- 六、避難方向指示燈之箭頭指向，應與避難路徑一致。

##### 第 146-5 條

下列場所應使用 A 級或 B 級之出口標示燈及避難方向指示燈：
- 一、車站、航空站、捷運車站、高速鐵路車站。
- 二、地下建築物。
- 三、觀光旅館、百貨商場、超級市場、遊藝場及其他類似場所。
- 四、總樓地板面積在一千平方公尺以上之百貨商場、超級市場、展覽場、集會堂及其他類似場所。
- 五、無開口樓層之地下層，其樓地板面積在一千平方公尺以上者。

前項場所之出口標示燈，其光度應在二十燭光（cd）以上；避難方向指示燈，其光度應在二十五燭光（cd）以上。

##### 第 146-6 條

1. 戲院、電影院、演藝場、集會堂、體育館、音樂廳、觀覽場及其他類似場所之觀眾席部分，應設置觀眾席引導燈。
2. 前項觀眾席引導燈，應為綠色燈光，標示觀眾席通道之前後兩端及中間位置，並應符合下列規定：
    - 一、裝設於通道地板上，其照度在零點二勒克司（Lux）以上。
    - 二、燈具為防水型且不易破損之構造。
    - 三、引導燈配置之間距在五公尺以下。

##### 第 146-7 條

出口標示燈及避難方向指示燈應保持不熄滅。但有下列情形之一者，得予減光或消燈：
- 一、場所無人在場期間。
- 二、設有自動火警報警設備或自動撒水設備之場所，其由自然採光之出入口或窗戶直接辨識避難方向者。
- 三、劇院、電影院、演藝場觀眾席部分，於放映或演出時，因業務需要由管理權人切換減光或消燈，且於火災發生時能自動點亮者。
- 四、旅館、醫院、養護機構、長期照顧機構、安養機構等供人住宿之場所，其客房內之睡眠區域，於就寢時間得予減光或消燈。

##### 第 153 條

1. 避難指標，應設於避難路徑之必要處所，以箭頭及避難方向標示之。
2. 前項避難指標之設置，應符合下列規定：
    - 一、設置位置應不妨礙通行，且其周圍不得設置影響視線之廣告招牌或其他物品。
    - 二、避難指標之大小、顏色、亮度及設置高度，依 CNS 10208 之規定。
    - 三、自出入口至避難指標之步行距離在七點五公尺以下，且易於識別該避難方向者。
    - 四、設於觀眾席引導燈附近之避難指標，應配合該引導燈設置。

##### 第 154 條

出口標示燈、避難方向指示燈及避難指標，應依中央主管機關之認可基準辦理，並標示 CNS 10208 之規定。

##### 第 155 條

1. 出口標示燈及避難方向指示燈應連接緊急電源，其容量應能於停電時，持續供電二十分鐘以上。
2. 設於地下層、無開口樓層或十一層以上樓層之主要避難路徑，其出口標示燈及避難方向指示燈之緊急電源容量，應能持續供電六十分鐘以上。
3. 前項緊急電源應使用蓄電池，其充電方式應符合 CNS 10208 之規定。

##### 第 156 條

出口標示燈及避難方向指示燈之配線，除依第二章之規定外，並應符合下列規定：
- 一、配線應為專用回路，並不得與其他設備共用。
- 二、緊急電源回路之配線，應施予耐燃保護或使用耐燃電纜。
- 三、配線應依屋內線路裝置規則之規定辦理。

#### 第二節 避難器具

##### 第 157 條

1. 各類場所應依下表選擇設置避難器具：

| 場所類別 | 應設避難器具 |
|---------|------------|
| 一、戲院、電影院、演藝場、歌廳、集會堂、體育館、音樂廳、觀覽場及其他類似場所 | 避難梯、緩降機、救助袋、滑臺、避難橋、避難繩索或滑杆 |
| 二、學校、補習班、訓練班 | 避難梯、緩降機、救助袋或滑臺 |
| 三、醫院、療養院 | 緩降機、救助袋或滑臺 |
| 四、長期照顧機構、安養機構、老人福利機構、護理機構 | 緩降機、救助袋或滑臺 |
| 五、托嬰中心、幼兒園、兒童及少年安置教養機構 | 避難梯、緩降機、救助袋或滑臺 |
| 六、旅館、觀光旅館、招待所 | 緩降機、救助袋或滑臺 |
| 七、飲食店、餐廳、咖啡廳、茶室 | 緩降機、救助袋或避難梯 |
| 八、百貨商場、超級市場、商場 | 緩降機、救助袋或滑臺 |
| 九、工廠、作業場 | 避難梯、緩降機、救助袋或滑臺 |
| 十、寺廟、教堂、宗祠 | 緩降機、救助袋或避難梯 |
| 十一、辦公室 | 緩降機、救助袋或避難梯 |
| 十二、集合住宅 | 緩降機、救助袋或避難梯 |
| 十三、其他經中央主管機關公告之場所 | 依公告規定設置 |

2. 前項各場所選擇設置之避難器具，應依其收容人員、建築物構造、樓層高度及避難路徑之條件，選擇適當之器具。

##### 第 158 條

前條各場所設置之避難器具，符合下列規定之一者，得予減設：
- 一、建築物主要構造為防火構造，且設有二座以上不同方向之直通樓梯者，各層之避難器具數量得減為應設數量之二分之一。
- 二、建築物各層設有可通達避難層或地面之陽台，且設有緩降機或救助袋者，該層得減為應設數量之二分之一。
- 三、建築物樓層高度在十公尺以下者，該層之避難器具數量得減為應設數量之二分之一。

##### 第 159 條

各類場所符合下列各款規定之一者，得免設避難器具：
- 一、建築物為防火構造，設有二座以上不同方向之直通樓梯，且各戶（間）有可通達該直通樓梯之陽台或走廊者。
- 二、建築物之各層樓地板面積在二百平方公尺以下者。
- 三、建築物各部分之樓層高度均在十公尺以下，且各層均設有可通達避難層或地面之陽台，並設有緩降機或救助袋者。
- 四、設有符合規定之特別安全梯或緊急升降機間之排煙室者。
- 五、其他經中央主管機關認可得免設或緩設者。

##### 第 160 條

1. 避難器具之設置數量，應依下表計算收容人員後，按每五十人（含未滿）設置一具。但各類場所之收容人數計算基準，應依下表規定：

| 場所類別 | 收容人員計算基準 |
|---------|----------------|
| 一、戲院、電影院、演藝場、集會堂、體育館、音樂廳、觀覽場 | 一、固定席位者：以該席位數計算之。<br>二、立位者：以樓地板面積每零點三平方公尺核算一人。 |
| 二、歌廳 | 一、客席部分：以樓地板面積每二平方公尺核算一人。<br>二、舞臺及其他部分：以樓地板面積每三平方公尺核算一人。 |
| 三、學校、補習班、訓練班 | 一、教室：以樓地板面積每二點五平方公尺核算一人。<br>二、辦公室及其他部分：以樓地板面積每四平方公尺核算一人。 |
| 四、醫院、療養院 | 一、病房：以病床數核算。<br>二、門診及其他部分：以樓地板面積每四平方公尺核算一人。 |
| 五、長期照顧機構、安養機構、老人福利機構、護理機構 | 一、寢室：以床位數核算。<br>二、其他部分：以樓地板面積每四平方公尺核算一人。 |
| 六、托嬰中心、幼兒園、兒童及少年安置教養機構 | 一、保育室、活動室：以樓地板面積每三平方公尺核算一人。<br>二、其他部分：以樓地板面積每四平方公尺核算一人。 |
| 七、旅館、觀光旅館、招待所 | 一、客房：以客房數每間核算二人。<br>二、員工及公共部分：以樓地板面積每四平方公尺核算一人。 |
| 八、飲食店、餐廳、咖啡廳、茶室 | 一、客席部分：以樓地板面積每二平方公尺核算一人。<br>二、廚房及其他部分：以樓地板面積每四平方公尺核算一人。 |
| 九、百貨商場、超級市場、商場 | 以樓地板面積每四平方公尺核算一人。 |
| 十、工廠、作業場 | 以樓地板面積每四平方公尺核算一人。 |
| 十一、寺廟、教堂、宗祠 | 以樓地板面積每二平方公尺核算一人。 |
| 十二、辦公室 | 以樓地板面積每四平方公尺核算一人。 |
| 十三、集合住宅 | 以各住宅單元之居室數每室核算一人。 |

2. 居室面向設有陽台者，其陽台於計算容納人數時，得扣除陽台面積。
3. 各層樓地板面積之計算，不包括樓梯間、電梯間、機械室、廁所、浴室及其他類似處所。

##### 第 161 條

避難器具之裝設，應符合下列規定：
- 一、設於避難時易於接近之處所。
- 二、與安全梯、特別安全梯及直通樓梯等避難路徑之距離，在步行距離三十公尺以下。
- 三、設於樓地板高度在十公尺以上之樓層時，應優先設置緩降機或救助袋。
- 四、設於具防火構造之陽台或露台時，其陽台或露台之面積應在四平方公尺以上。
- 五、避難器具之設置位置，不得妨礙避難通路及防火門之開啟。

##### 第 162 條

避難器具之開口部，其開口面積應符合下表規定：

| 避難器具種類 | 開口面積 |
|------------|---------|
| 緩降機       | 高六十公分以上、寬六十公分以上 |
| 滑臺         | 高八十公分以上、寬八十公分以上 |
| 避難橋       | 高八十公分以上、寬一百公分以上 |
| 救助袋（斜降式）| 高八十公分以上、寬八十公分以上 |
| 救助袋（直降式）| 高六十公分以上、寬六十公分以上 |
| 避難梯       | 高八十公分以上、寬六十公分以上 |
| 滑杆         | 直徑八十公分以上之圓孔或八十公分平方以上 |
| 避難繩索     | 高六十公分以上、寬六十公分以上 |

| 滑杆         | 一公尺×一公尺以上 |

##### 第 164 條

避難器具之下降空間，應符合下表規定：

| 避難器具種類 | 下降空間 |
|------------|---------|
| 緩降機       | 無障礙物之垂直下降空間，水平方向半徑五十公分以上 |
| 滑臺         | 無障礙物之傾斜下降空間，寬度一公尺以上 |
| 避難橋       | 無障礙物之水平延伸空間，寬度一公尺以上 |
| 救助袋（斜降式）| 無障礙物之傾斜下降空間，寬度一公尺以上 |
| 救助袋（直降式）| 無障礙物之垂直下降空間，水平方向半徑六十公分以上 |
| 避難梯       | 無障礙物之傾斜下降空間，寬度六十公分以上 |
| 滑杆         | 無障礙物之垂直下降空間，水平方向半徑五十公分以上 |
| 避難繩索     | 無障礙物之垂直下降空間，水平方向半徑五十公分以上 |

##### 第 165 條

避難器具之下降空地，應符合下表規定：

| 避難器具種類 | 下降空地面積 |
|------------|------------|
| 緩降機       | 二平方公尺以上 |
| 滑臺         | 四平方公尺以上 |
| 避難橋       | 二平方公尺以上（對岸） |
| 救助袋（斜降式）| 四平方公尺以上 |
| 救助袋（直降式）| 二平方公尺以上 |
| 避難梯       | 四平方公尺以上 |
| 滑杆         | 二平方公尺以上 |
| 避難繩索     | 二平方公尺以上 |

前項下降空地，應符合下列規定：
- 一、設於地面，且無障礙物。
- 二、不得設於排水溝、屋頂突出物或其他有安全疑慮之處所。
- 三、與建築物牆面之距離在五十公分以上。

##### 第 166 條

避難器具之標示，應依下列規定辦理：
- 一、避難器具設置位置應有標示，並標明使用方法。
- 二、設置位置標示之顏色應為紅色或綠色，且具蓄光或自發光功能。
- 三、使用方法之說明文字，應以中文標示，其字體大小在零點五平方公分以上。
- 四、自居室任一點至避難器具設置位置之指標，應設於避難通路之必要處所。

##### 第 167 條

緩降機之構造，除應符合 CNS 13204 之規定外，並應符合下列規定：
- 一、機體須為金屬製或同等以上強度之不燃材料。
- 二、捲揚器（調速器）應設於容易操作之位置，且調速性能良好。
- 三、繩索應為直徑八毫米以上之鋼索或具同等以上強度及耐久性者，並設有防止扭轉之裝置。
- 四、安全帶應為寬五十毫米以上之帶狀構造，且附有安全扣環。
- 五、緩降機之落下速度，應在每秒十六公分以上、一百五十公分以下，且具有自動調速功能。
- 六、收納箱為不燃材料或以厚度在零點五毫米以上之鋼板製造。
- 七、緩降機之固定基座，應以 M12 以上之膨脹螺栓固定。

##### 第 168 條

滑臺之構造，應符合下列規定：
- 一、主體為金屬製或具同等以上強度及耐久性者。
- 二、斜度在四十五度以下。
- 三、表面平滑且無突出物。
- 四、寬度在五十公分以上，且深度在十公分以上之溝狀或具二側護欄。
- 五、滑臺之出口端應設有減速裝置或緩衝裝置。
- 六、固定部分應以螺栓固定，且其固定基座為堅固之鋼筋混凝土構造。

##### 第 169 條

避難橋之構造，應符合下列規定：
- 一、主體為金屬製或具同等以上強度及耐久性者。
- 二、寬度在五十公分以上。
- 三、兩端應設有安全固定裝置，且與建築物之連結部分，應具充分之強度。
- 四、設有護欄或扶手，其高度在七十五公分以上。
- 五、具水平延伸功能者，其伸展及收納應動作確實。

##### 第 170 條

救助袋之構造，應符合下列規定：
- 一、袋體為具防火性能之纖維材料或同等以上性能者。
- 二、袋體內部應為螺旋狀或設有減速裝置。
- 三、斜降式救助袋之斜度在四十五度以下。
- 四、直降式救助袋之內部直徑在六十公分以上。
- 五、進口框架為金屬製或具同等以上強度者，且設有安全扶手。
- 六、袋體出口端應設於地面或避難層。
- 七、收納箱應為不燃材料或以厚度在零點五毫米以上之鋼板製造。

##### 第 171 條

避難梯之構造，應符合下列規定：
- 一、主體為金屬製或具同等以上強度及耐久性者。
- 二、寬度在四十公分以上。
- 三、梯級之間距在三十公分以下，且具有防滑功能。
- 四、斜度在六十度以下。
- 五、設有護欄或扶手，其高度在七十五公分以上。
- 六、固定部分應以螺栓固定，且其固定基座為堅固之構造。

##### 第 172 條

滑杆及避難繩索之構造，應符合下列規定：
- 一、滑杆：
    - （一）主體為金屬管，直徑在四十五毫米以上。
    - （二）表面平滑且無接縫或突出物。
    - （三）下端應設有緩衝裝置或避難空地。
    - （四）固定基座應以螺栓固定於堅固之構造體。
- 二、避難繩索：
    - （一）繩索為直徑十五毫米以上之麻繩、棉繩或具同等以上強度者。
    - （二）繩索兩端應設有安全掛鉤或固定裝置。
    - （三）繩索長度應能到達地面或避難層。
    - （四）收納箱應為不燃材料。

##### 第 173 條

避難器具之支固器具及材料，應符合下列規定：
- 一、材料應為鋼材或不銹鋼，並符合 CNS 2473（一般結構用軋鋼料）及 CNS 4435（一般結構用碳鋼鋼管）之規定。
- 二、鋼材表面應施予防銹處理。
- 三、支固器具之強度，應能承受該避難器具之最大負荷重量之二倍以上。
- 四、鋼材之接合部分，應以熔接或同等以上強度之方法接合。

##### 第 174 條

避難器具之螺栓固定，應符合下列規定：
- 一、固定用螺栓之直徑及數量，依下表規定：

| 避難器具最大使用人數 | 螺栓直徑 | 每處最少螺栓數 |
|-------------------|---------|-------------|
| 一人               | M10     | 二支         |
| 二人以上未滿五人    | M12     | 四支         |
| 五人以上           | M16     | 六支         |

- 二、螺栓應為不銹鋼或經防銹處理者。
- 三、螺栓埋入深度，在混凝土構造應在五十毫米以上；在磚造或木造構造，應以貫通方式固定並加墊板。
- 四、膨脹螺栓應符合 CNS 5114 之規定。
- 五、螺栓間距應在十五公分以上。

#### 第三節 緊急照明設備

##### 第 175 條

緊急照明設備之構造，應符合下列規定：
- 一、光源應為白熾燈、螢光燈（日光燈）、水銀燈、發光二極體（LED）或其他經中央主管機關認可者。
- 二、燈具應使用不易破損及耐熱之材料。
- 三、燈具周圍溫度應在攝氏四十度以下。
- 四、緊急照明燈應附有充電指示燈及點燈試驗裝置。
- 五、設有防止觸電之措施，且接地應符合屋內線路裝置規則之規定。

##### 第 176 條

緊急照明設備之配線，除依第二章之規定外，並應符合下列規定：
- 一、電源回路應直接連接於分路，且不得裝設插座或開關。
- 二、配線應施予耐燃保護或使用耐燃電纜。
- 三、與其他設備共用分路時，該分路之過電流保護器容量，應能承受緊急照明燈全部點燈時之電流。

##### 第 177 條

1. 緊急照明設備應連接緊急電源，其容量應能於停電時，持續供電三十分鐘以上。
2. 前項緊急電源為蓄電池時，其容量應能持續供電三十分鐘以上；為自備發電機與蓄電池組合時，蓄電池容量應能持續供電十分鐘以上，且自備發電機能於停電後三十分鐘內自動啟動供電。
3. 前項蓄電池之充電方式，應符合 CNS 8802 之規定。

##### 第 178 條

緊急照明設備之照度，應符合下列規定：
- 一、地下通道、地下層、無開口樓層：樓地板面之照度在十勒克司（Lux）以上。
- 二、前款以外之場所：樓地板面之照度在二勒克司（Lux）以上。
- 三、避難路徑轉折處或高低差處：樓地板面之照度在十勒克司（Lux）以上。

前項照度，應於緊急電源供電狀態下量測之。

##### 第 179 條

下列處所得免設緊急照明設備：
- 一、主要為通往室外之出入口，且有自然採光，其照度可達十勒克司（Lux）以上者。
- 二、洗手間、浴室、機械室、鍋爐房及其他類似處所。
- 三、自居室任一點易於識別該居室出入口，且有自然採光者。
- 四、其他經中央主管機關公告之場所得免設。
